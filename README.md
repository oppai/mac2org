mac2org
=======

this is shell command to find organization from MAC address

### How to use
  
  $ mac2org [MACADDRESS(any expression)]

### Sample

	$ mac2org 00-05-02-00-00-00
	000502     (base 16)		APPLE COMPUTER

	$ mac2org 00:05:02
	000502     (base 16)		APPLE COMPUTER

	$ mac2org 000502000000
	000502     (base 16)		APPLE COMPUTER
